# Oi, eu sou o Cesar! 👋

Sou um **Analista de Dados e Financeiro** apaixonado por otimização de processos e arquitetura de dados. Meu trabalho consiste em converter grandes volumes de informação bruta em dashboards intuitivos e automações que geram eficiência real para o negócio.

### 📶 O que eu faço
* **Migração de Dashboards:** Especialista em traduzir lógicas complexas de Excel para ambientes robustos em Power BI (DAX).
* **Automação de Dados:** Desenvolvimento de scripts para limpeza de dados e integração de relatórios financeiros e logísticos.
* **Análise Preditiva e Gerencial:** Foco em indicadores de performance (KPIs), revisão de custos (KM/Combustível) e saúde financeira.

### 🛠️ Tech Stack 
* **Dados:** SQL, Python, Power BI, Advanced Excel.
* **Metodologia:** Conhecimentos em Programação Orientada a Objetos (POO) e modelagem de dados relacional.
* **Workflow:** Gestão de conhecimento através de **Obsidian** (Smart Note-taking) para documentação de processos técnicos.

### 🎓 Desenvolvimento Contínuo
* **Google Data Analytics Professional Certificate:** Aprofundando competências em análise estatística e visualização de dados.
* **Sistemas & Hardware:** Entusiasta de arquitetura de sistemas e configurações de hardware para alta performance.
